# Custom UI fonts

Drop `.ttf` or `.otf` files here to make the theme configurator prefer them.

Recommended names:

- `display.ttf` or `display.otf` for large headings
- `ui.ttf` or `ui.otf` for interface text
- `mono.ttf` or `mono.otf` for monospace text

If these aliases are absent, the first font file in this folder is used as the
UI/display family. The file stem is treated as the font family name, so a file
named `Manrope.ttf` should contain a family named `Manrope`.
